Little Boxes

Little Boxes is an interactive digital art installation created in a group as part of a Human Computer Experience Design Studio. It is a reflection on history. It aims to reconcile personal narratives to the reporting of events; intermingling statuses, photos and tweets with news, movies and songs of the past. Various wooden boxes of different sizes, shapes and designs are scattered through the space, each with their own screen stored inside.

Unified Public Transport System for Sydney

A group project involving the redesign of Sydney's public transport ticketing system and user interfaces. We produced designs for a smart card system, a mobile app, a ticketing machine and tickets with a major focus on improving the user experience for both experienced and novice users.

Our group came in 1st place, recognised by Cubic.

Multitouch

This multitouch touchtable application allows the user to explore the world's highest grossing movies of all time. The user is able to see how much each movie grossed, various connections between the series, the movies' trailers as well as some other information.

The app was built in Processing, using the TuioZones, oscP5 and GSVideo libraries, with the data being imported from an XML file.

Unicef Mobile Site

I designed a website for Unicef optimised for mobile devices. The brief wanted us to make it as easy as possible for users to donate on the go, as well as sign up for a newsletter and learn more about the top 10 causes.

My design was chosen as the best in the class, in terms of mobile optimisation.
